
Установка и запуск: 
	Бэкэнд:
		1. Установить интерпретатор Python3 с сайта (https://www.python.org/downloads)
		2. В папке проекта выполните в терминале: python ./start.py
		3. Проверьте работу в браузере по адресу: http://127.0.0.1:23456/api
		* таблицы базы можно посмотреть программой "DB Browser for SQLite"(https://sqlitebrowser.org/)
	Фронтэнд:
		1. В папке проекта перейдите в каталог "vue".
		2. Выполнить в терминале: npm install
